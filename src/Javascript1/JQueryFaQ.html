FAQ


  • 1 ¿Por qué es importante una sección de FAQ o preguntas frecuentes?

    • Funciona para mejorar el SEO
    • Ahorros en atención al cliente
    • Gestión del tiempo
    • Fideliza una compra

  • 2 Ejemplos de preguntas frecuentes en una página web

    • ¿Qué beneficios tiene adquirir este producto o servicio?
    • ¿Realmente necesito este producto o servicio?
    • ¿Cómo funciona?
    • ¿Tiene garantía?
    • ¿Cuánto dinero cuesta la adquisición de este producto?
    • ¿Qué métodos de pago aceptan?
    • ¿En cuánto tiempo tendré listo mi producto?
    • ¿Qué productos están disponibles y qué productos no?
    • ¿Cómo empiezo?

  • 3 Consejos para crear una página de preguntas frecuentes

    • No utilices este espacio para promocionarte
    • Agrega un buscador
    • No crees demasiadas preguntas
    • Organiza tus preguntas por orden de frecuencia
    • Agrega todos los elementos que necesites para darte a entender
